\section{Fonts}
\code{lmodern} is the default font used in this document and is defined in
\code{styles/tw-scrbook-template-main.sty}. On \href{https://texwelt.de/wissen/fragen/5537/was-macht-eigentlich-usepackaget1fontenc}
{\code{texwelt.de}} there is also a good (german) description about the
\code{fontenc} package.
\begin{sscodeenv_default}{%
    style=tst-listing-tex,%
    label=lbl_font_packages,%
    caption=Fonts packages}
    \RequirePackage[T1]{fontenc}
    \RequirePackage{lmodern}
\end{sscodeenv_default}
